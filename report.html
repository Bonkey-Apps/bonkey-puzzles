Report a Bug or Request a Feature

Found something broken? Have an idea to make Bonkey Puzzles better? We'd love to hear from you.

Open a GitHub Issue (preferred)

The fastest way to get a bug fixed or a feature considered is to open an issue on GitHub. It takes about a minute and your report goes straight to the development queue.

Report a Bug on GitHub →

Request a Feature on GitHub →

No GitHub account? Email us instead.

That's fine — just send an email to bonkey.apps@gmail.com with a description of the bug or idea. Please include:

  • Which game were you playing?
  • What difficulty level?
  • What happened, and what did you expect to happen?
  • Your device model and Android version (if reporting a bug).
