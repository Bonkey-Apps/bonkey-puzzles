Report a Bug

Found something wrong? We want to know!

Found a bug or want to suggest a feature? Choose the option that works best for you — both reach us directly.

What to include in your report

  • The puzzle type and difficulty level
  • Your device model (e.g. Samsung Galaxy S22)
  • OS version (e.g. Android 14 / iOS 17)
  • App version (found in Settings → About)
  • Steps to reproduce the problem
  • A screenshot if possible

Open a GitHub Issue

Preferred — lets us track the issue and keep you updated. Requires a free GitHub account.
